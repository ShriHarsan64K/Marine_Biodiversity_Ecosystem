## ⚙️ Step-by-Step Environment Setup (Ubuntu / WSL2)

### Step 1 — System Update

```bash
sudo apt update && sudo apt upgrade -y
sudo apt install -y git curl wget build-essential
```

### Step 2 — Python 3.11

```bash
sudo apt install -y software-properties-common
sudo add-apt-repository ppa:deadsnakes/ppa -y
sudo apt update
sudo apt install -y python3.11 python3.11-venv python3.11-dev

# Verify
python3.11 --version   # Expected: Python 3.11.x
```

### Step 3 — Virtual Environment

```bash
cd ~
python3.11 -m venv marine_venv
source marine_venv/bin/activate

# Add to ~/.bashrc so it activates automatically:
echo 'source ~/marine_venv/bin/activate' >> ~/.bashrc
```

### Step 4 — CUDA 11.8 (WSL2 specific)

```bash
# For WSL2 — use the wsl-ubuntu installer (NOT the native Linux one)
wget https://developer.download.nvidia.com/compute/cuda/repos/wsl-ubuntu/x86_64/cuda-wsl-ubuntu.pin
sudo mv cuda-wsl-ubuntu.pin /etc/apt/preferences.d/cuda-repository-pin-600

wget https://developer.download.nvidia.com/compute/cuda/11.8.0/local_installers/cuda-repo-wsl-ubuntu-11-8-local_11.8.0-1_amd64.deb
sudo dpkg -i cuda-repo-wsl-ubuntu-11-8-local_11.8.0-1_amd64.deb
sudo cp /var/cuda-repo-wsl-ubuntu-11-8-local/cuda-*-keyring.gpg /usr/share/keyrings/
sudo apt update
sudo apt install -y cuda-11-8

# Add to PATH
echo 'export PATH=/usr/local/cuda-11.8/bin:$PATH' >> ~/.bashrc
echo 'export LD_LIBRARY_PATH=/usr/local/cuda-11.8/lib64:$LD_LIBRARY_PATH' >> ~/.bashrc
source ~/.bashrc

# Verify
nvcc --version   # Expected: release 11.8
```

### Step 5 — PyTorch 2.0 (CUDA 11.8)

```bash
pip install torch==2.0.1+cu118 torchvision==0.15.2+cu118 torchaudio==2.0.2+cu118 \
    --index-url https://download.pytorch.org/whl/cu118

# Verify
python -c "import torch; print(torch.cuda.is_available()); print(torch.cuda.get_device_name(0))"
# Expected: True  |  NVIDIA GeForce RTX 3050
```

### Step 6 — All Other Dependencies

```bash
pip install -r requirements.txt
```

### Step 7 — Kaggle API Setup

```bash
# 1. Go to https://www.kaggle.com/settings → API → Create New Token
# 2. Copy kaggle.json to the right place:
mkdir -p ~/.kaggle
cp /path/to/downloaded/kaggle.json ~/.kaggle/
chmod 600 ~/.kaggle/kaggle.json

# Verify
kaggle datasets list -s fish
```

## 🚀 Running Phase 1 (Step-by-Step)

```bash
# Navigate to project root
cd ~/marine_biodiversity_project
source ~/marine_venv/bin/activate

# ── Phase 1A: Setup ──────────────────────────────────────────────────────────
python scripts/phase1a_scaffold_project.py       # Create workspace dirs
python scripts/phase1a_verify_installation.py    # Must show 7/7 ✅

# ── Phase 1B: Data Collection ────────────────────────────────────────────────
python src/utils/phase1b_map_species.py          # Build species → class mapping
python src/utils/phase1b_download_roboflow.py    # Download Roboflow datasets
python src/utils/phase1b_download_kaggle.py      # Download Kaggle datasets

# ── Phase 1C: Data Cleaning ──────────────────────────────────────────────────
# Preview duplicates first (safe)
python src/utils/phase1c_remove_duplicates.py --input_dir dataset/raw --dry_run
# Then actually remove them:
python src/utils/phase1c_remove_duplicates.py --input_dir dataset/raw

# Quality filter (preview)
python src/utils/phase1c_filter_quality.py --input_dir dataset/raw --dry_run
# Then actually filter:
python src/utils/phase1c_filter_quality.py --input_dir dataset/raw

python src/utils/phase1c_validate_annotations.py \
    --input_dir dataset/raw --output_dir dataset/standardized
python src/utils/phase1c_generate_statistics.py --dataset_dir dataset/standardized

# ── Phase 1D: Augmentation & Split ───────────────────────────────────────────
python src/utils/phase1d_analyze_balance.py \
    --dataset_dir dataset/standardized --target 800
python src/utils/phase1d_augment_dataset.py \
    --input_dir dataset/standardized --output_dir dataset/augmented
python src/utils/phase1d_split_dataset.py \
    --input_dir dataset/augmented --output_dir dataset/processed

# ── OR run everything automatically ─────────────────────────────────────────
bash scripts/run_phase1.sh
```
